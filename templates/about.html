About the app


What?

Philly Wheels is a web application for finding wheelchair accessible routes via SEPTA, Philadelphia's public transit system. All stops on the resulting route are equipped with ramps and elevators and meet ADA standards for wheelchair accessibility.

Why?

Elli Balcomb built this app for her final project at Hackbright Academy. Elli has a good friend in Philly that uses a wheelchair and gave her the idea! Philadelphia was an ideal city for this project because it is one of the largest cities in the country that does not provide this crucial tool for people with disabilities.

How?

Built using Python, Flask, Javascript, SQLAlchemy, and Jinja. The app does the following:

Provides place predictions for start and end addresses via the Google Places Autocomplete API
Translates addresses into latitude and longitude coordinates via the Google Geocoding API
Determines the closest accessible public transit stops via the SEPTA API
Calculates the shortest route via an implementation of Dijkstra's Graph Search Algorithm
